# Nightbase Official Site

This repository contains the Next.js 16 marketing site for Nightbase.

## Getting Started

Install dependencies and start the development server:

```bash
npm install
npm run dev
```

Then open [http://localhost:3000](http://localhost:3000) to view the site locally.

## Available Scripts

- `npm run dev` – Starts the Next.js development server.
- `npm run build` – Builds the production application.
- `npm run start` – Runs the built app in production mode.
- `npm run lint` – Lints the codebase using the Next.js ESLint configuration.

## Project Structure

```
app/
  layout.tsx      # Shared layout, metadata, and navigation
  page.tsx        # Landing page content
  globals.css     # Global styles for the marketing site
public/
  favicon.svg     # Vector favicon for the site
```

The project uses the Next.js App Router and React Server Components by default.
