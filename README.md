# NightBase Official Website

NightBase is an Apple-inspired marketing site for the NightBase SaaS platform built with Next.js 14 App Router, TypeScript, Tailwind CSS, shadcn/ui primitives, and Framer Motion animations.

## Getting Started

```bash
pnpm install
pnpm dev
```

The site is organized in the `/src/app` directory following the App Router conventions and provides both Japanese (`/`) and English (`/en`) experiences.

## Features

- Hero section with glassmorphism design and motion effects
- Comprehensive feature, pricing, case study, and legal pages
- Responsive layout powered by Tailwind CSS
- Shared navigation and footer components with locale switching
- Contact form posting to a Next.js route handler for easy integration with email/CRM providers

## Tech Stack

- Next.js 14 (App Router)
- TypeScript
- Tailwind CSS & shadcn/ui tokens
- Framer Motion
